### 五行
乾三连，坤六断。  
震仰盂，艮覆碗。  
离中虚，坎中满。  
兑上缺，巽下断。

乾为天、坤为地、震为雷、巽为风、艮为山、兑为泽、坎为水、离为火。

五行相生的次序是：金生水，水生木，木生火，火生土，土生金。

五行相克的次序是：金克木，木克土，土克水，水克火，火克金。
